Agency Pro README
Agency Pro Portfolio | Cedar & Code
The ultimate high-end template for boutique studios and high-ticket freelancers.

What’s in the Folder:

- index.html: The high-impact Bento Grid layout.
- style.css: Advanced typography and grid system.
- script.js: Handles the smooth scroll and reveal animations.

Customization:

1. Typography: Best with Inter and Playfair Display.
2. Bento Grid: Replace background-image styles with project shots.
3. Services: Horizontal layout for high-ticket descriptions.
